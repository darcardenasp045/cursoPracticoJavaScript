Inicio Cuadrado Triangulo Circulo

Este proyecto es mi primer proyecto de desarrollo web, en el cual se muestran las formulas para calcular el area y perimetro de las figuras geometricas.

En la parte superior se encuentran los enlaces para acceder a las paginas de las figuras geometricas.

En cada pagina se encuentra un formulario para ingresar los datos necesarios para calcular el area y perimetro de cada figura.

En la parte inferior se encuentra el resultado de los calculos.

En la parte superior se encuentra el enlace para regresar a la pagina de inicio.
